Network-Wide Ad Blocking [Pi-Hole]

What's A Pi-Hole?

Can I eat one?

Sadly, Pi-Holes aren't edible confections. That doesn't mean that they aren't sweet though! One summer I was getting more and more fed up with all of the ads that I was seeing day after day everywhere I looked. While I do understand that ads are a service that allow for many free online services, many ad companies use extremely invasive tactics to target people with not only ads but also specific online media. Eventually enough was enough, and I bought 2 raspberry-pis with the intent to set them up as house-wide ad blockers.
A Pi-Hole is basically a custom DNS server.

What's a DNS?

DNS Server GraphicDNS stands for "Domain Name Service". When someone tries to look up information online, they will usually type words in to a search bar [i.e. "Foobar"]. The DNS server takes the human-readable input and converts it into a machine-readable IP address. Once the device has the address, it can ask the web server for the information stored at that IP address.

So why do I need a custom DNS server?

By setting up a custom DNS server, you gain the ability to blacklist or block specific IP addresses. This is useful because many websites don't actually host their own ads. Instead, many ads are stored on large physical servers. Thankfully, the task of figuring out what the IP addresses of those physical servers are isn't your job. There are many "blocklists" that are maintained by communities of people who hate ads and trackers enough to spend their free time hunting down and publishing these IP addresses. By taking advantage of these online blacklists, it's easier than ever to start blocking ads and trackers on your home network.

That's cool! How does a raspberry pi become a DNS server?

Pi-Hole OverviewThe raspberry pi is given a private IP address from the router. By making this a fixed address you can have other devices on that network use the rasperry pi's IP as a DNS server so that all the DNS queries will be sent to the raspberry pi. Once all the traffic is being routed through the raspbeery pi, it can start blocking requests that are blacklisted.
